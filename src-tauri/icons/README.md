# Wallet Icon Configuration for Multiple Platforms

This directory should contain icons for different platforms:

## Required Icons

### Desktop (Windows, Linux, macOS)
- `32x32.png` - Small icon for taskbar
- `128x128.png` - Standard desktop icon  
- `128x128@2x.png` - High DPI desktop icon
- `icon.ico` - Windows icon file
- `icon.icns` - macOS icon file

### Mobile (Android, iOS)
- `icon-48.png` - Android launcher icon (mdpi)
- `icon-72.png` - Android launcher icon (hdpi)
- `icon-96.png` - Android launcher icon (xhdpi)
- `icon-144.png` - Android launcher icon (xxhdpi)
- `icon-192.png` - Android launcher icon (xxxhdpi)

### PWA (Web)
- `icon-192.png` - PWA icon (manifest)
- `icon-512.png` - PWA icon (manifest)

## Icon Guidelines

- Use the Chert logo or a simplified wallet icon
- Maintain consistent branding across platforms
- Follow platform-specific design guidelines
- Ensure icons are crisp at all sizes
- Use transparency where appropriate

For now, placeholder icons can be used until proper branding assets are created.
